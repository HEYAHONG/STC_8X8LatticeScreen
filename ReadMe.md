# 说明

使用STC15W204S驱动点阵屏的工程。

## 目录说明

- DOC:数据手册等文档。
- FW:固件代码。采用mcu8051ide开发。
- HW:硬件代码。采用立创ide开发。

# 工具

## 软件开发

mcu8051ide直接在ubuntu下可使用以下指令安装:

```bash
sudo apt-get install mcu8051ide
```

## 硬件开发

参见 https://lceda.cn

## 烧录

使用 stc-isp 下载,请在[STC官网]( http://www.stcmcu.com/)下载。。
